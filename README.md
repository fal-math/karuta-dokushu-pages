# 読手ツール (Karuta Assistant Tool)

競技かるたの読手を支援するWebベースの補助ツールです。

## 主な機能

- **柔軟な練習モード**:
  - 全100首を順番に読むモード。
  - 練習したい札のみを自由に選択して読むモード。
- **インテリジェントなタイマー**:
  - かるたの読みのリズム（下の句〜余韻〜間合い〜上の句）を視覚的に表示する円グラフタイマー。
  - 1ユニットの秒数を細かく調整可能。
- **進捗の自動保存**:
  - `localStorage` を使用し、現在の札や選択状態、設定を自動的に保存。
  - ブラウザを閉じたりリロードしたりしても、すぐに続きから再開できます。
- **離脱防止機能**:
  - 不意のブラウザ操作（戻るボタンなど）による練習の中断を防ぐ警告ダイアログ。
- **モダンな技術構成**:
  - Vanilla JS (ES Modules) による軽量な動作。
  - JSDoc による型定義で、保守性と開発効率を向上。

## ファイル構成

- `index.html`: エントリポイント
- `src/css/`: スタイルシート（変数、メインレイアウト、コンポーネント）
- `src/js/`: ロジック
  - `main.js`: 全体の初期化とイベント制御
  - `timer.js`: タイマーの計算・描画
  - `cardRender.js`: 歌詞（縦書き・ルビ）の描画
  - `storage.js`: localStorage の操作
  - `clock.js`: 現在時刻の表示
  - `types.js`: JSDoc 用の型定義
- `src/data/`: 歌データ (JSON)

## 使い方

1. アプリを起動するとスタート画面が表示されます。
2. 「全100首」または「札を選択」を選んで練習を開始します。
3. 練習画面では「スタート」でタイマーが開始されます。
4. メニューからいつでもスタート画面に戻り、設定を変更したり札を選び直したりできます。
