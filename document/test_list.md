# テスト一覧

現在、以下の自動テスト（Playwright）が定義されています。
ファイル: `tests/checks.spec.js`

## Picker UI Tests

札選択画面（Picker UI）の動作検証を行うテスト群です。

| テスト名                                     | 説明                                                                                                                     |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| Sorting by ID (Default)                      | 初期表示のアコーディオンが歌番号（ID）順にソートされていることを確認します。                                             |
| Sorting by Kimariji (50-on)                  | ソート順を「五十音順」に変更した際、並び順が変化することを確認します。                                                   |
| Filtering by first character (あ)            | 「あ」行フィルタを適用した際、「あ」で始まる札のみが表示・選択状態になることを確認します。                               |
| Filtering by length (1 char)                 | 「1字決まり」フィルタを適用した際、該当する札のみが選択状態になることを確認します。                                      |
| Filtering by Tens place (0) including ID 100 | 歌番号の十の位が「0」（1桁および100番）のフィルタを適用した際、該当する札（1〜9, 100）が選択状態になることを確認します。 |

## App Flow Tests

アプリの進行（フロー）に関する動作検証を行うテスト群です。
ファイル: `tests/flow.spec.js`

| テスト名                  | 説明                                                                                                         |
| ------------------------- | ------------------------------------------------------------------------------------------------------------ |
| Initial State             | アプリ初期状態（1/100, 前へボタン無効化など）が正しいことを確認します。                                      |
| Navigate Next and Prev    | 「次へ」「前へ」ボタンによるカード遷移と、進捗表示（X / 100）の更新、ボタン有効/無効状態を確認します。       |
| Exit to Start             | メニューから「終了する」を選択し、確認ダイアログを経てスタート画面に戻る挙動を確認します。                   |
| Timer Reset on Navigation | 計測中にカード移動（「次へ」）を行った際、タイマーがリセットされ、計測ボタンが有効化されることを確認します。 |

## テストの実行方法

以下のコマンドを実行することで、ローカルサーバーの起動からテスト実行、終了までを自動で行います。

```bash
npm test
```

※ テスト実行中はポート **3333** を使用します。

## テスト結果（レポート）の確認方法

テストが失敗した場合や、詳細なログを確認したい場合は、以下のコマンドでHTMLレポートを開くことができます。

```bash
npx playwright show-report
```
